# Installation

1. Download the latest release from Cfx.re Assets
2. Extract the files to your server's resources folder
3. Import the SQL file to your database
4. Add `start redutzu-mdt-esx` to your server.cfg
5. Enjoy the script!

# Requirements

- [es_extended]
- [oxmysql/mysql-async]
- [screenshot-basic]
